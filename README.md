Daily Activity Tracker

The Daily Activity Tracker is a web-based application designed to help users plan, monitor, and manage their day-to-day tasks efficiently. The system allows users to create a list of daily activities and track their completion status using an intuitive checkbox interface.

Users can add, update, and delete activities for each day, and mark tasks as completed once finished. The application maintains a structured record of daily progress, enabling users to stay organized, build productive habits, and improve time management.

The frontend is developed using HTML, CSS, JavaScript, and Bootstrap to ensure a responsive and user-friendly interface. The backend is built with PHP Laravel, providing robust routing, authentication, and data handling. A SQL database is used to securely store user activities and their completion status.

Key features include:

Create, edit, and delete daily tasks
Mark tasks as completed using checkboxes
Track activities on a day-wise basis
Responsive and interactive user interface
Secure data storage and management using Laravel and SQL

This project demonstrates full-stack development skills, including frontend design, backend logic, and database integration, while solving a real-world productivity problem.
